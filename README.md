# tiny-world-java
Tiny World created with Java and OpenGL for Computer Graphics class Fall 2022.


Controls are: WASD and mouse

To run the code:
1. Clone the repo
2. Open in IntelliJ IDEA and wait for Gradle to sync
3. Run the am.aua.game.Game class.
